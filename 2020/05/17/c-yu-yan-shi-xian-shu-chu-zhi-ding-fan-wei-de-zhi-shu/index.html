新版本：主函数中实现

思路：
1.大于2的偶数都不是质数，只对奇数进行整除判断
2.只整除小于等于no的平方根的质数//由长方形面积和约数

#include

int *prime(int n){
    int prime[100000]={0};//数组元素个数可以扩展
    int ptr =0;
    prime[ptr++]=2;
    prime[ptr++]=3;
    for(int no=5;no<=n;no+=2){//大于2的偶数都不是质数
        int flag=0;

        for(int i=1;prime[i]*prime[i]<=no;i++){//只整除小于等于no的平方根的质数
            if (no%prime[i]==0) {
                flag=1;
                break;
            }
        }
        if(!flag)
            prime[ptr++]=no;
    }

    return prime;
}
