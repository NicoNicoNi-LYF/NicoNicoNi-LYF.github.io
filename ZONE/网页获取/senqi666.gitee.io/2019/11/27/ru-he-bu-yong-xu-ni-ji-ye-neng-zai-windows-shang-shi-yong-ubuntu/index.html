不用虚拟机也能在Windows上使用Ubuntu—Windows Terminal 微软新版终端工具安装教程

不知道大家是否有在虚拟机内使用 Ubuntu 卡到想口吐芬芳的感觉。如果你不想给你的电脑做第二个系统的话可以尝试一下跟着教程安装~~

1. 查看电脑版本号

​ 按住快捷键 “win+i” 打开 Windows设置，选择 “系统” ，找到底部的 “关于” ， 查看 “Windows规格” 内的 “版本号”。

2. 更新系统(请阅读下方内容确定是否符合要求)

​ 如果版本在 “1903” 以下(1903用户可以忽略此步骤)，请访问下方网址进行更新，否则安装 Windows Terminal 时会报错。Windows更新工具会自动更新最新版本，不用担心数据丢失，放心~~

​ PS：博主今天刚刚从 “1803” 更到 “1909”，自己真的是八辈子没有更新过……

​ Windows10自动更新工具

3. 开启相关配置

  • 开启适用于 Linux 的 Windows 子系统选项

    找到开始菜单边上的搜索图标，搜索 “控制面板”，打开控制面板：

​

​ 点击 “卸载程序”：

​

​ 点击 “启用或关闭 Windows 功能”：

​ 滚动条拉到底部，找到 “适用于 Linux 的 Windows 子系统” 并选择，点击 “确定”，并重启电脑：

  • 打开 “开发人员模式”

    按住 “win+i” 快捷键，打开 Windows 设置，选择 “更新和安全”：

​ 点击 “开发者选项”：

​ 选择 “开发人员模式”：

​ 接下来需要你稍等片刻，这需要一段时间。请等待选项下方提示完成。等待时间内可以先进行下一步的压缩包下载，后续步骤请等待安装组件完毕。

3.安装 Windows Terminal

​ 速速戳下方链接下载压缩包解压~

​ Windows Terminal

4. 安装证书

首先安装证书到受信任人

​ 如果你开启了 “开发人员模式”，可以根据下方步骤进行操作。PS：只有点击了列表内的元素，详细信息才会亮起：

​ 点击 “详细信息” 后，选择 “查看证书”：

​ 点击 “安装证书”：

​ 选择 “本地计算机”，并点击 “下一步”：

​ 进入 “证书向导”，依照下图的步骤进行操作。选择 ”将所有的证书都放入下列存储“，点击 ”浏览“ ，选择 ”受信任人“，最后点击 ”确定“：

​ 点击 ”下一步“ ，之后点击 ”完成“，系统会弹出 ”导入成功“。

5. 安装 Windows Terminal

​ 由于博主已经安装过，所以不显示安装按钮，根据下方截图提示，并双击安装即可：

​ 安装完成后自动弹出 Windows Terminal，表示安装成功：

6. Windows Terminal 相关操作：

​ 摁住 ”ctrl+t“，会生成第二个cmd窗口，此时右上角会出现 ”倒三角“ 标志。点击它，可以进入 Power Shell 和 Settings(设置)：

​

​ Windows Terminal 是开源的。可以点击 ”Settings“，进入设置，会打开 ”profiles.json“ 文件，修改 Json数据可以进行美化，具体自行百度。

6. 安装Ubuntu

​ 在开始菜单中，找到 Microsoft Store：

​ 右上角搜索 ”ubuntu“ 并点击下载，由于博主已经安装过，这里显示的是启动：

​ 安装完Ubuntu后，由于是 WSL 版的 Linux，所以还需要开启 WSL 的权限。找到开始菜单边上的搜索，输入 power，以 ”管理员身份运行“：

​ 然后输入如下命令：

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

​ 等待命令执行完毕，在开始菜单内找到 ”Ubuntu“并打开，会显示如下界面，表示正在安装配置，等待一段时间：

​ 安装完毕后会提示你创建用户名和密码。

​ 注意：输入之前最好将输入法切换成Windows自带输入法，用户名只输入小写字母。

​ 输入完毕后就可以去 Windows Terminal 里验证一下啦~~

7. 验证并使用Ubuntu

​ 接下来我们就可以去 Windows Terminal里玩转 Ubuntu 啦！！快打开你的 Windows Terminal 输入 ubuntu 试一下吧：

​ 注意：刚安装完Ubuntu，用户目录是没有文件的。python 和 pip 还需要各位伙伴们根据命令自行安装哦~~

Ubuntu命令行下查看 Windows 分区里面的文件，输入以下命令即可：

cd /mnt

Ubuntu 在 Windows 上的位置是：

C:\Users\你的用户名\AppData\Local\Packages\CanonicalGroupLimited.UbuntuonWindows_79rhkp1fndgsc\LocalState\rootfs

以上两条希望可以给你一些参考。教程在此完毕啦，快去体验吧~~
